# Button Sizing Guide

---

## Medium Button (48px)

| Property           | Value            |
|:-------------------|:-----------------|
| Overall Height     | 48px             |
| Label Type         | Label - Medium   |
| Text Size          | 16px             |
| Icon Size          | 24×24px          |
| Padding (Left)     | 12px             |
| Padding (Right)    | 12px             |
| Padding (Top)      | 12px             |
| Padding (Bottom)   | 8px              |
| Corner Radius      | 8px              |

---
